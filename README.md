# Cardomancer

Гра Cardomancer на React Native з Expo.

## Встановлення

1. Встановіть залежності:
```bash
npm install
```

2. Запустіть проект:
```bash
npm start
```

## Структура проекту

- `App.js` - головний компонент додатку
- `app.json` - конфігурація Expo
- `package.json` - залежності та скрипти
